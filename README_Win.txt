    INSTALLATION INSTRUCTIONS
    ----------------------------------------------------------------
    1. Locate the installer file:
       LogicXplorer_Setup.exe

    2. Double-click the installer to start setup.

    3. Follow the on-screen wizard:
       • Click "Next" to continue
       • Choose installation directory (default recommended)
       • Select optional shortcuts (Desktop / Start Menu)

    4. Click "Install" to begin installation.

    5. Once completed, click "Finish" to exit the installer.

    ----------------------------------------------------------------
    UNINSTALLATION
    ----------------------------------------------------------------
    You can safely remove the application using any of the following:

    Method 1: Windows Settings
    • Settings → Apps → Installed Apps → LogicXplorer → Uninstall

    Method 2: Start Menu
    • Start Menu → LogicXplorer → Uninstall

    Method 3: Control Panel
    • Control Panel → Programs → Uninstall a program
    • Select LogicXplorer → Uninstall

    All installed files and shortcuts are removed automatically.
